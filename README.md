## 🔷 InterviewAI — Smart Interview Simulator

### 🧩 What it does

InterviewAI is a futuristic platform that prepares users for real interviews using AI-based simulations, performance tracking, and resume intelligence.

### 🚀 Key Highlights

* Real-time **AI interview simulation** (video + voice)
* **Resume ATS scoring** with improvement suggestions
* Intelligent **performance feedback system**
* Company-oriented **mock interview practice**
* Interactive dashboard with analytics

### 💡 Why it matters

Helps candidates reduce anxiety, improve communication, and perform better in real interview scenarios.
